2. Name: `GP2040-CE-Mayflash-S5`
3. Set to **Public**

### Step 4: Push to GitHub (5 minutes)
The setup script will guide you with exact commands to run

### Step 5: Build Firmware (10 minutes)
1. GitHub repository → **Actions** tab
2. Select **Build Firmware** workflow
3. Click **Run workflow**
4. Wait for build to complete
5. Download the `.uf2` file artifact

1. Hold **BOOTSEL** button on Pico
2. Connect USB to computer
3. Drag `.uf2` file onto **RPI-RP2** drive
4. Board reboots automatically ✅

**Total time: ~30 minutes**

## 🆘 Common Questions

**Q: Do I have to use GitHub?**
A: The easiest way is GitHub Actions (it does the build for you). You can also build locally if you have CMake, Pico SDK, and ARM GCC installed.

**Q: Can I modify button mapping?**
A: Yes! After flashing, connect to the web configurator. Go to http://192.168.4.1

**Q: What if the build fails?**
A: Check the Actions tab for error messages. Most common issues:
- Repository not set to Public
- Missing files (shouldn't happen with our package)
- GitHub Actions not enabled (default is enabled)

**Q: Can I use a different board?**
A: Yes, change the `GP2040_BOARDCONFIG` parameter in GitHub Actions to any board in the `configs/` folder.

**Q: Will this work on other consoles?**
A: Yes! The firmware still supports PC, Switch, PS4, Xbox, etc. Just change the input mode via web UI or button shortcuts.
